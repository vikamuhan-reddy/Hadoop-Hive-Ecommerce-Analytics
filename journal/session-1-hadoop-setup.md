# Session 1 - Hadoop Cluster Setup

## Objective

Create a Hadoop environment for learning HDFS and Hive.

## Activities

- Installed Docker
- Fixed Docker permission issues
- Learned difference between docker compose and docker-compose
- Started Hadoop cluster

## Verified Services

- NameNode
- DataNode
- ResourceManager
- NodeManager
- HistoryServer

## Commands Used

docker compose up -d

docker ps

jps

## Key Learnings

- Docker permissions often cause startup failures.
- Containerized Hadoop is easier than manual installation.
- Understanding service roles helps troubleshooting.

## Outcome

Working Hadoop Cluster.
